# The-GameBox
This repository contains all information about The GameBox including all project files of the demo game AstroShot inspired by Space Invaders. It is required to have a GameBox to play AstroShot, however there is a build for keyboard input gameplay

Paths for AstroShot Builds:
    1. Keyboard Input:
        AstroShot Builds\AstroShot Keyboard Version Installer.exe

To Edit AstroShot in the Unity game engine:
    1. Pull from this repository to your local disk
    2. Open Unity Hub
    3. Add 
    4. Add project from disk
    5. Select AstroShot folder in this repository
    6. Edit however you'd like
